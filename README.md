# xjyToDoList
徐嘉一的基于Django ,Django Rest Framework,react的ToDoList

# 项目结构

1. todolist-react: react客户端
2. todolist-django: django服务端

# install

1. cd todolist-django
2. python manage.py runserver 端口号
